# 📘 Técnicas de Programação e Algoritmos

Repositório dedicado à disciplina de **Técnicas de Programação e Algoritmos**, contendo exercícios, implementações e estudos realizados ao longo do curso.

## 📚 Sobre a disciplina

A matéria aborda os principais conceitos fundamentais da programação, com foco em:

- Lógica de programação  
- Estruturas de controle (condicionais e loops)  
- Estruturas de dados básicas  
- Algoritmos e sua análise  
- Resolução de problemas computacionais  

## 🧠 Conteúdos abordados

- ✔️ Variáveis e tipos de dados  
- ✔️ Estruturas condicionais (`if`, `else`, `switch`)  
- ✔️ Estruturas de repetição (`for`, `while`, `do while`)  
- ✔️ Vetores e matrizes  
- ✔️ Funções  
- ✔️ Ponteiros (se estiver usando C)  
- ✔️ Algoritmos de ordenação e busca  
- ✔️ Manipulação de dados  

## 💻 Linguagem utilizada

- C / C++ (ajuste conforme necessário)

## 📂 Estrutura do repositório
    
- `listas/` → listas e práticas da aula  
- `provas/` → resoluções de provas ou simulados  

## 🚀 Objetivo

Este repositório tem como objetivo:

- Praticar lógica de programação  
- Melhorar a resolução de problemas  
- Servir como material de estudo futuro  
- Acompanhar a evolução ao longo da disciplina  

## 📌 Observações

- Os códigos podem conter versões diferentes de solução para o mesmo problema  
- Alguns exercícios podem estar comentados para facilitar o entendimento  
- Repositório em constante atualização 📈
