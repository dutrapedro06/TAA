# Técnicas de Análise de Algoritmos (TAA)

Repositório desenvolvido durante a disciplina de **Técnicas de Análise de Algoritmos (TAA)** do curso de Ciência da Computação.

O projeto reúne listas de exercícios, desafios práticos e avaliações com foco no desenvolvimento de lógica de programação, análise de algoritmos, estruturas de dados e resolução eficiente de problemas utilizando as linguagens **C** e **C++**.

---

## 🚀 Tecnologias Utilizadas

- Linguagem C
- Linguagem C++
- GCC / G++
- Visual Studio Code
- Linux / WSL

---

## 📚 Conteúdos Estudados

Durante a disciplina foram implementados diversos algoritmos envolvendo:

- Análise e desenvolvimento de algoritmos
- Complexidade computacional
- Manipulação de bits (Bitwise)
- Operadores:
  - AND
  - OR
  - XOR
  - Shift de bits
- Recursividade
- Vetores e matrizes
- Ponteiros
- Manipulação de memória
- Estruturas de dados
- Pilhas
- Filas
- Árvores
- Algoritmos de busca
- Algoritmos de ordenação:
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
- Problemas matemáticos
- Programação competitiva

## ⚙️ Como Executar

Clone o repositório:

```bash
git clone https://github.com/dutrapedro06/TAA.git
```

Acesse a pasta:

```bash
cd TAA
```

### Compilando arquivos C:

```bash
gcc arquivo.c -o programa
```

Execute:

```bash
./programa
```

### Compilando arquivos C++:

```bash
g++ arquivo.cpp -o programa
```

Execute:

```bash
./programa
```
